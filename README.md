# A Simple BabylonJS Example with TypesScript

This is a simple example of BabylonJS with TypesScript.

You can see the result [here](https://corysia.github.com/basic).

## How to build

---

### Software Requirements

- npm
- Typescript

### Install dependencies

```bash
npm install
```

### Build

```bash
npm run build
```
or
```bash
npm run serve
```
